## Этап 2. PHP

  

### 2.0 Общие основы (обязательно перед ООП — пробелы не только в классах)

- [ ] Пройти целиком [Хекслет — «Основы PHP для начинающих»](https://ru.hexlet.io/programs/php-basics-free) (54 урока)


Отдельно проверить/закрыть по ходу или после курса:

- [ ] Суперглобальные переменные: `$_GET`, `$_POST`, `$_SESSION`, `$_SERVER`

- [ ] `try/catch/finally`, `throw` — обработка ошибок вне контекста классов

- [ ] Функции для работы с массивами: `array_map`, `array_filter`, `usort` и т.п.

- [ ] PSR-12 — включить автоформатирование по PSR в IDE, не учить как теорию

- [ ] Отладка: `var_dump`/`print_r`, что такое Xdebug

  

### 2.1 ООП (после 2.0)

По порядку:

- [ ] [«PHP: Введение в ООП»](https://ru.hexlet.io/courses/php-introduction-to-oop) — классы, интерфейсы, инкапсуляция

- [ ] [«PHP: Погружаясь в классы»](https://ru.hexlet.io/courses/php-classes) — трейты, абстрактные классы, позднее связывание

- [ ] [«PHP: Объектно-ориентированный дизайн»](https://ru.hexlet.io/courses/php-object-oriented-design) — паттерны, SOLID на практике

- [ ] Service Container / Dependency Injection

- [ ] Eloquent-отношения: hasOne, hasMany, belongsTo, belongsToMany (на своём примере User → Order)
